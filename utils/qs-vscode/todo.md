## Whichkey
- [-] Action
  - Find and replace
    - "r" Current File
    - "R" Project
  - "l" / "L" Live Server
- [-] Code / LSP
  - r Rename
- [-] Toggle + UI
  - "a" Toggle tool/ activity bar
  - "b" Toggle Sidebar
  - "n" File Explorer
  - "p" Panel (terminal,output,etc)
  - "f" fullscreen
  - "s" Statusbar/Modeline
  - "M" Menubar
  - "T" Tab
  - "z" Zen
- [ ] Debug
- [ ] Terminal
